# voice cloning prototype

This project demonstrates few-shot voice cloning using neural text-to-speech models.

### Features:

• Voice cloning from short audio samples  
• Neural text-to-speech generation  
• Interactive UI  
• Runtime performance tracking  
• Download generated speech  

### Technologies used:

• Python  
• PyTorch  
• Coqui TTS  
• Gradio  

### Pipeline:

Voice sample -> Speaker conditioning -> Neural TTS -> Generated speech

### Future Improvements:

• Voice similarity scoring  
• Frequency analysis  
• Noise removal  
• Multi-speaker database  
• Real-time microphone cloning
